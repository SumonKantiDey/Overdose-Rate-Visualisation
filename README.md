# Drug Overdose Deaths Analysis

Below are the key files and where to find the exploratory data analysis (EDA) and the cleaned dataset used by the project.

## Preview
[View Live Demo](https://raw.githack.com/SumonKantiDey/Overdose-Rate-Visualisation/main/M4/index.html)

## Important files and locations

- EDA/preprocessing notebook: `Data Preprocessing.ipynb`
- Cleaned data (processed): `data/clean_drug_overdose_death.csv`

## Code Usage Instructions

First, clone this repo and move to the directory. Then, install the necessary libraries.

```bash
git clone https://github.com/SumonKantiDey/Overdose-Rate-Visualisation.git
cd Overdose-Rate-Visualisation/
```

## How to run M4 locally

After setup, run a local server in the project folder:

```bash
$ python -m http.server 5500
```

Or if you have VS Code, use the Live Server extension:
- Right-click on `death_vs_health_facilities/index.html`
- Select "Open with Live Server"

Then open `http://localhost:5500` in your browser
