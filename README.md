# Drug Overdose Deaths Analysis

Below are the key files and where to find the exploratory data analysis (EDA) and the cleaned dataset used by the project.

## Important files and locations

- EDA/preprocessing notebook: `Data Preprocessing.ipynb`
- Cleaned data (processed): `data/clean_drug_overdose_death.csv`

## How to run M4 locally
1. Clone this repository
2. In the project folder, run a local server:
   - If you have Python: `python -m http.server 5500`
   - If you have VS Code: use the Live Server extension
3. Open `http://localhost:5500` in your browser
